== Description ==

Kandinsky is the Block Theme designed to help you create a personal website that looks like 'bento.me' using WordPress. Create your personal single page website by clicks!

== Installation ==

1. Make sure you're running WordPress 6.5 or later.
2. Upload the theme.
3. Activate the theme.
